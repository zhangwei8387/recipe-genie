# RecipeGenie Backend

This is the backend for RecipeGenie, built with FastAPI and Microsoft Agent Framework.

## Structure

- `app/`: Main application code
  - `api/`: API route definitions (REST + AG-UI)
  - `core/`: Core configuration, security, and utilities
  - `models/`: SQLModel database models
  - `agents/`: AI Agent definitions and workflows
  - `main.py`: Application entry point
- `tests/`: Test suite
