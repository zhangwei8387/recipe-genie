## 如何使用本索引

- 点击目录链接可以在仓库中快速跳转到对应文档或模板。  
- 在开始撰写任何正式文档前，请先在对应目录下复制模板并填写（例如：从 `requirements/SRS_template.md` 开始）。

## 贡献指南（如何提交文档）

1. 创建新文档或更新模板时，请在 `docs/` 下对应目录中创建或修改文件。  
2. 使用清晰的文件名和合适的标题（中文/英文皆可）。  
3. 在 PR 描述中注明变更摘要、关联的 issue/任务和验收标准。  
4. 文档更新应包含简短的变更日志（例如：`文档版本 v1.1 - 更新 API 接口规范`）。
