# Sprint 1 - 基础设施（Week 3-4）

## 目标
搭建项目脚手架及开发环境，为后续迭代打好基础。

## 任务列表
- [ ] 项目脚手架搭建（项目模板、目录结构） — 估时：1 天
- [ ] 开发环境配置（本地运行、环境变量说明） — 估时：1 天
- [ ] 基础 UI 框架（组件库 + 样式系统） — 估时：2 天
- [ ] 数据库初始化（schema、迁移工具） — 估时：1 天
- [ ] CI 基础配置（GitHub Actions / GitLab CI） — 估时：1 天

## 验收标准
- 可以在本地一键启动开发环境
- CI 能够在 PR 上运行基本 lint 与测试

## 备注
请在每项任务后附带负责人与预计完成日期。
