# 软件需求规格说明书（SRS）模板

## 1. 项目概述
- 项目名称：RecipeGenie
- 版本：v1.0
- 目标用户：健康饮食者、上班族等
- 项目概述：一个 AI 实时交互式食谱推荐平台。

## 2. 术语与缩写
- AI: 人工智能
- SRS: 软件需求规格说明书

## 3. stakeholders（利益相关者）
- 产品经理
- 开发团队
- UI/UX 设计师
- 测试团队
- 运维

## 4. 功能需求
- 功能 1：食谱推荐
  - 描述：基于用户偏好和可用食材给出个性化推荐
  - 优先级：高
  - 验收标准：用户输入约束条件后 3 秒内返回 5 个候选食谱

## 5. 非功能需求
- 性能：推荐延迟 < 500ms（目标）
- 可用性：99.9% 可用性目标
- 安全：用户数据加密、合规性说明
